# Documents (PDFs)

Drop your PDFs here to have them appear (with a live preview + download link) on the **CV & Resume** page:

| File path        | Shows as                  |
|------------------|---------------------------|
| `files/cv.pdf`     | "↓ Curriculum Vitae (PDF)" |
| `files/resume.pdf` | "↓ Resume (PDF)"           |

The filename must match exactly. To change the labels or add more documents, edit the `docs:[ … ]` array on the CV entry in `Saturn Orrery.dc.html`.
